Cronómetro App

Una aplicación simple para medir el tiempo de manera eficiente y funcional.

Características

  • Fácil de usar.
  • Interfaz limpia y clara.
  • Funciones de inicio, pausa y reinicio del cronómetro.
  • Funcionalidad de contar vueltas, ideal para entrenamientos y competiciones.
  • Incluye temporizador con configuración de tiempos personalizados.
